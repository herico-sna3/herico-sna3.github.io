# Grupo Herol — Site Institucional

## Descrição

Site institucional do Grupo Herol, holding que reúne a **Herol Tecnologia** (matriz, divisão de tecnologia e inovação) e a **AeroMap Engenharia** (divisão de engenharia e mapeamento com drones).

## Design

- Tema escuro premium (dark mode) com fundo navy `#0a0f1e`
- Destaques em ciano elétrico (`#00d4ff`) e verde tech (`#10b981`)
- Glassmorphism, animações sutis e efeitos glow
- Totalmente responsivo

## Tecnologias

- HTML5
- CSS3 (variáveis, flexbox, grid, animações)
- JavaScript (menu mobile, scroll reveal)
- Ícones em SVG inline
- Zero dependências externas

## Estrutura do Site

- **Header** fixo com navegação
- **Hero** com stats (+2 divisões, +850 projetos, +12 anos)
- **Divisões**: Herol Tecnologia + AeroMap Engenharia
- **AeroMap Engenharia**: serviços de Topografia, Modelagem 3D e Inspeção Aérea
- **Sobre o Grupo**: história e pilares (Segurança, Precisão, Inovação, Equipe)
- **Footer** com contato e redes sociais

## Como usar

1. Faça o download do `index.html`
2. Crie um repositório no GitHub (ex: `grupo-herol`)
3. Faça upload do arquivo
4. Ative o GitHub Pages em **Settings → Pages** (branch `main`, pasta `/root`)

## Personalização

Para alterar cores, edite as variáveis CSS no `:root`:

- `--primary`: cor de destaque (ciano)
- `--accent-green`: tom de destaque secundário
- `--bg-deep`: cor de fundo principal
